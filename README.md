# Bootstrap-project
Project Overview

This project is a three-page responsive website built using Bootstrap 5 as part of the UI development task. The goal of the project was to explore Bootstrap components, understand their structure, and remix them into clean, modern, and user-friendly web pages. The site includes a Home page, About/Services page, and Contact page, each structured with reusable Bootstrap components.

How I Completed the Project (Step-by-Step Process)
1. Studying Bootstrap Documentation

I started by visiting the official Bootstrap documentation and browsing through the Examples section.
I explored components like:

Navbar

Carousel

Cards

Buttons

Forms

Modal

Grid system

This helped me understand how each component works and how they can be combined.

2. Collecting UI Components

I reviewed multiple example templates from Bootstrap and selectively picked components that fit the website's structure. Instead of copying entire pages, I extracted only useful building blocks such as:

Navbar layout

Hero section

Auto-sliding carousel (data attributes only)

Card-based feature section

Modal for additional information

Responsive contact form

3. Designing and Building the Pages

I structured the project using three HTML files:

#Home Page (index.html)
Includes:

Navbar

Hero section with text + carousel

A modal trigger button

Sliding carousel (HTML-only)

Features/cards section

Responsive layout

Footer

#About Page
Contains:

Service explanation

Information sections

Cards & accordion components

#Contact Page
Contains:

Fully responsive contact form

Map iframe section

Bootstrap form controls

I ensured the spacing, layout, and component combination looked modern and visually appealing.

4. Ensuring Responsiveness

I tested the website on:

Desktop

Tablet sizes

Mobile screen widths

Bootstrap’s grid and utilities automatically handled most responsiveness, and minor adjustments were made for spacing and alignment.

5. Adding Final Touches

I refined:

Color tones

Shadow effects

Rounded corners

Buttons and spacing

Modal content

Footer year script

These cosmetic adjustments improved the overall UI feel.

6. Hosting the Project

I uploaded all files to a GitHub repository, committed the code, and then deployed the project using GitHub Pages (or Netlify, depending on preference).
This allowed the project to have a live URL that could be accessed from any device.

Some of the challenges included:

Understanding how Bootstrap’s grid adjusts at different breakpoints

I solved these by testing frequently, reading the Bootstrap docs, and adjusting the CSS/HTML as needed.
